PÁGINA DE PRUEBA

Esta propuesta consiste en un desarrollo para consulta web de información del reporte entregado en formato excel, por medio de angular y typescript.

Para el caso planteado, la data se publico en una base de datos MySql y por medio de PHP se consulta la base de datos y se constuye la estructura usada por AMCHARTS para la graficación animada.

El segundo modelo esta construido en Microsoft Power BI, utilizando la misma data y algunas herramientas de visualización propuetas que incluyen algunas ventajas de interactividad.

Se disponen al costado derecho de esta página el acceso a las secciones construidas. Por algunas fallas al momento de exportar fue necesario mantener todos los componente temporalmente en una sola página.
